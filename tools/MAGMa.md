## Description
MS Annotation based on in silico Generated Metabolites (MAGMa) automatically annotates MS^n data, where n â‰¥ 2, without requiring reference spectral data. Substructures are derived from in silico hierarchical trees. Assignments at each MS level take both the previous as well as the successive fragments into account. Candidate structures are ranked by scores. MAGMa automatically searches PubChem and HMDB.
